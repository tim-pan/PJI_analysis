All CSV files record the performance of each model parameter setup in each dataset.
CNN_result_smote.csv records the performance of SM dataset.
CNN_result_smote_pca.csv records the performance of SM+PCA dataset.
CNN_result_rs.csv records the performance of RS dataset.
CNN_result_rs_pca.csv records the performance of RS+PCA dataset.

draw_fig.ipynb visualize the outcome of each csv file

CNN_weight folder records the parameter file for each setup of each dataset.

tr.xlsx and test_final.xlsx is the final training and testing datasets respectively.
