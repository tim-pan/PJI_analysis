All CSV files record the performance of each model parameter setup in each dataset.
LR_result_smote.csv records the performance of SM dataset.
LR_result_smote_pca.csv records the performance of SM+PCA dataset.
LR_result_rs.csv records the performance of RS dataset.
LR_result_rs_pca.csv records the performance of RS+PCA dataset.

draw_fig.ipynb visualize the outcome of each csv file

tr.xlsx and test_final.xlsx is the final training and testing datasets respectively.
